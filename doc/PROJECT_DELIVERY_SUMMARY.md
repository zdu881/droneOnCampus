## 🚀 快速使用指南

### 1. 访问测试工具
```
URL: http://localhost:8080/droneOnCampus/test_light_control.html
功能: 完整的灯光控制测试和演示
```

### 2. 基础 API 调用
```javascript
// 改变灯光颜色
await ueApiManager.changeBaseStationLight(1, 0);  // 灯光1设为红色(0)

// 快速操作
await ueApiManager.setBaseStationGreen(0);   // 全部绿色
await ueApiManager.setBaseStationRed(1);     // 灯光1红色
await ueApiManager.setBaseStationYellow(2);  // 灯光2黄色

// 高级效果
await ueApiManager.blinkBaseStationLight(1, 0, 3, 300);  // 灯光1闪烁

// 状态自动映射
await ueApiManager.setBaseStationStatusLight(1, "detecting");  // 黄色
```

### 3. 与检测系统集成
在基站运维检测中添加灯光反馈：
- 检测开始 → 黄灯（processing）
- 检测成功 → 绿灯（success）
- 检测失败 → 红灯（error）

```
地址: http://10.30.2.11:30010/remote/object/call
方法: PUT
头部: Content-Type: application/json
```

```

```

### 颜色映射
```

1 = 绿色 (Normal/Idle)
2 = 黄色 (Warning/Processing)
```
